# Title
Interactive Movies  Application 

## 📌 Objective
An interactive and responsive Movie Application built using ReactJS.  
Users can browse movies, view detailed movie information,  and explore trending Movie.

## Tech stack
Frontend - React JS (React Router, Redux or React Context API, CSS or CSS frameworks, Git, and GitHub for hosting the repository.)

#  Technologies Used

- React JS
- React Router DOM
- JavaScript
- CSS3
- OMDB API
- js-cookie
- React Icons

# 📌 Objective

Build a ReactJS application that allows users to:

- Browse movies
- View movie details
- Explore trending and popular movies

#  Features

-  Login Authentication
-  Home Page with Movie Banner
-  Funny Movies Section
-  All Movies Listing
-  Trending Movie Details
-  Fully Responsive Design
-  Modern Netflix-like UI
-  API Integration using OMDB API
-  JWT Token Authentication using Cookies
